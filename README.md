# English

**Capture image URLs effortlessly & beautifully.**

ImageURLcpy is a refined Chrome extension that lets you easily batch capture image URLs from any webpage, with custom formatting and instant clipboard copy.

## âœ¨ Features

- **Batch Capture**: One-click to start, continuously capture multiple image URLs.
- **Smart Highlight**: Hover over images to see a translucent green overlay.
- **Instant Copy**: Click to copy URL instantly.
- **Custom Format**: Set prefix & suffix for single or multiple URLs (Markdown, HTML, etc.).
- **Live Preview**: See exactly how your formatted URLs will look.
- **Elegant Design**: Morandi green palette, glassmorphism, smooth animations.
